# Initalisierung

Dieses Crate wurde mit folgenden Befehlen erstellt.

```bash
mkdir -p src/Rust
cd src/Rust
cargo new --bin pirates-party
cd pirates-party
git add .
git commit -a -m "Initaler Commit"
git remote add origin git@github.com:zzeroo/pirates-party.git
git push -u origin master
```
